## M-pesa-Payment
- An android application that consumes safaricom Daraja API to facilitate Customer to Business Payment(C2B)

## Prerequisite
- Android Studio
- API-KEY -> CONSUMER KEY, CONSUMER SECRET

## Tech stack
- Java
- XML
- Retrofit - network calls
- Okhttp3
- GSON
- Okhttp Logging interceptor
